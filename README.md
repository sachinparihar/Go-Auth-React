# Go-Auth-React

## How to start

### Clone the repo

```
git clone https://github.com/sachinparihar/Go-Auth-React.git
```
```
docker-compose up
```

Paste the url: <http://localhost:3000/> in your browser

## To stop the Services

```
docker-compose down
```
